Jellyfin is the Free Software Media System that puts you in control of managing and streaming your media. It is an alternative to the proprietary Emby and Plex, to provide media from a dedicated server to end-user devices via multiple apps.

This repository contains scripts offered for public consumption on [our repository server](https://repo.jellyfin.org) so they may be managed from a central place.

## `install-debuntu.sh`

This script provides a quick install process for Debian and Ubuntu repository users. It is linked from the main server downloads page. The corresponding SHA256 checksum file **must** be updated after any changes.
